This is my interactive portfolio website showcasing my work as a backend developer and computer science student.  

It features:  
- **About Section:** Interactive 3D model to introduce myself.  
- **Projects Section:** Highlighting selected projects including AI, web, and mobile applications.  
- **Awards Section:** Hackathons and competitions I have participated in.  
- **Contact Section:** Quick links to email, LinkedIn, and GitHub.
